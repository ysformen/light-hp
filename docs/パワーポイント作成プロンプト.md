# パワーポイント作成用プロンプト（他AIへ）

以下をそのままChatGPTやGeminiなどに貼り付けて使用してください。

---

```
以下の打ち合わせ確認リストを、PowerPointのスライド資料にしてください。

【資料の目的】
葬儀社「市民葬のらいと」のホームページ制作にあたり、
担当者向けに費用・サービス・運用体制を説明するための打ち合わせ資料です。

【対象読者】
ホームページの管理・運用を担当する方（IT知識は多くないことを想定）

【スライド構成の指示】
1. 表紙：「市民葬のらいと ホームページ制作 打ち合わせ資料」
2. 目次
3. ドメインについて（概要・費用・選択肢）
4. ホスティング（サーバー）について（概要・費用・選択肢）
5. メール・お問い合わせフォームについて
6. 管理・運用体制について
7. セキュリティについて
8. アクセス解析について
9. 費用まとめ一覧（表形式）
10. 今後の確認事項チェックリスト

【デザイン指示】
- シンプルで見やすいデザイン
- 専門用語には補足説明を入れる
- 各スライドにポイントをまとめた吹き出しや注釈を入れる
- カラーは落ち着いた青系または白ベース

【内容】

## 1. ドメインについて
- 希望するドメイン名（例：raito-sougi.jp、light-sougi.com など）
- ドメインの種類（.jp / .com / .net など）
- ドメイン取得サービスの選定（お名前.com・ムームードメイン・Cloudflare など）
- 年間費用目安：.jp 約3,000〜4,000円／年、.com 約1,500〜2,000円／年

## 2. ホスティング（サーバー）について
- サーバーの選定（Vercel・Netlify・さくらインターネット など）
- 無料プランで運用するか、有料プランにするか
- 費用目安：Vercel無料プランは0円、有料プランは月$20〜
- 日本語サポートが必要かどうか

## 3. メール・問い合わせフォームについて
- 問い合わせ用メールアドレスの管理者確認
- お問い合わせフォームの送信先メールアドレスの確定
- フォーム送信サービスの選定（Formspree・SendGrid など）
- 費用目安：Formspree無料プランは月50件まで0円

## 4. 管理・運用体制について
- ページ内容を更新する担当者は誰か
- 更新頻度（お知らせ・料金変更 など）
- 更新作業を自分でするか、外部に依頼するか
- 緊急時（サイトが落ちた等）の連絡先

## 5. セキュリティ・SSLについて
- SSL（https化）の対応確認 → Vercel/Netlify なら自動で無料対応
- バックアップ体制の確認

## 6. アクセス解析について
- Google Analytics などの導入有無
- どのデータを確認したいか（アクセス数・問い合わせ数 など）
- 費用：Google Analytics は無料

## 7. 費用まとめ（目安）
| 項目               | 費用目安          | 頻度   |
|--------------------|-------------------|--------|
| ドメイン取得・更新 | 1,500〜4,000円    | 年1回  |
| ホスティング       | 0〜3,000円        | 月額   |
| フォーム送信       | 0〜2,000円        | 月額   |
| アクセス解析       | 0円               | −      |
| 合計（最小構成）   | 約1,500円〜       | 年間   |
```
